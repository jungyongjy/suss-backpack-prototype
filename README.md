# SUSS Backpack - UX/UI Redesign Prototype
**Module:** NCO205 (User-Centred Design)

This repository hosts a fully functional, mobile-optimized React prototype for a reimagined SUSS Backpack application. 

🔗 **[Click here to view the Live Interactive Prototype](https://github.com/jungyongjy/suss-backpack-prototype)**

## Project Overview
This prototype was developed following the Empathise, Define, Ideate, and Prototype phases of the Design Thinking process. It directly addresses user pain points identified through contextual inquiries by applying key Human Factors and Ergonomics (HFE) principles.

## Key Ergonomic Improvements
* **Reduced Cognitive Load:** Wayfinding was broken down into a 2-step process (3D Macro Building view -> 2D Micro Floor Plan) to better match human spatial mental models.
* **Improved Physical Ergonomics:** The library booking system was overhauled from tedious 30-minute lists to an intuitive 1-hour interval, tap-to-select grid.
* **Cognitive Offloading:** Added a "Sync to G-Cal" integration and a dynamic CGPA Simulator to shift complex tracking and calculations from the user's working memory to the system.
* **Pre-attentive Visual Processing:** Integrated live Crowd Cams utilizing color-coded occupancy states (Red/Green) for immediate visual evaluation without reading data.

## Technical Details
This prototype is built using React, Tailwind CSS, and Lucide Icons, compiled entirely in the browser via Babel Standalone.
